# Webpack-starter-boilerplate
A simple but powerful webpack boilerplate.

## Features:
- PostCSS
- Latest Webpack config
- SCSS / CSS
- Autoprefixer for SCSS and CSS
- Latest Babel config
- Typescript support
- Minification of CSS, HTML and JS
- Fonts support

## Yet to be added:
- Images optimizer
